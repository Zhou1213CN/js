Shell Program

This is the introdution for our operating system course's final project.

The goal of this project is implementing a shell program, that is, a program that repeatedly reads commands from the user, interprets them, and translates them into a sequence of actions that likely involve an interaction with the operating system through system calls. Basic features include cd, ls, background symbol, word count, and redirection.
